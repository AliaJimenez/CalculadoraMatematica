# Calculadora Matemática 🧮

Este repositorio contiene el desarrollo del **Primer Parcial (P1)** correspondiente a la materia de Programación I (Programacion Orientada a Objetos). El proyecto consiste en una aplicación de consola que implementa una calculadora matemática funcional utilizando programación estructurada y lógica condicional avanzada.

El proyecto fue desarrollado utilizando el IDE **NetBeans** y está configurado para compilarse con **Apache Ant**.

---

## 📋 Descripción del Proyecto

La aplicación procesa operaciones aritméticas y lógicas basadas en entradas del usuario. Cuenta con una estructura limpia dividida en componentes esenciales dentro del paquete `calculadoramatematica`:

*   **`CalculadoraMatematica.java`**: Clase principal que gestiona el flujo del examen, la captura de datos por consola, las estructuras de decisión (`switch`/`if-else`) y el procesamiento de los cálculos matemáticos.
*   **`string.java`**: Clase auxiliar utilizada para el manejo, validación o formateo especializado de cadenas de texto dentro de la lógica del parcial.

---

## 🚀 Funcionalidades Desarrolladas

*   **Procesamiento Aritmético:** Resolución de operaciones matemáticas básicas y complejas bajo requerimientos de examen.
*   **Validación de Entradas:** Manejo de excepciones lógicas o validaciones de texto mediante la clase auxiliar.
*   **Estructura Académica:** Código optimizado siguiendo las directrices de evaluación solicitadas en la materia.

---

## 🛠️ Tecnologías Utilizadas

*   **Lenguaje:** Java SE
*   **Entorno de Desarrollo (IDE):** NetBeans
*   **Sistema de Construcción:** Apache Ant (`build.xml`)
